# Aplicacoes-em-Java-2013
Conjunto de aplicações desenvolvidas em 2013, durante o início dos estudos com Java, utilizando o NetBeans, na faculdade.
O objetivo deste diretório é registrar a evolução prática com a linguagem, incluindo aplicações simples em console e interfaces gráficas.
Características
- Projetos criados no NetBeans
- Uso básico de orientação a objetos (classes, atributos e métodos)
- Aplicações em console e interfaces gráficas simples
- Estruturas iniciais de modelagem (ex: classes como Cliente, Carro, Atendente, etc.)
- Código com foco em aprendizado, sem preocupação com arquitetura ou boas práticas avançadas
Observações

Os códigos refletem um estágio inicial de aprendizado e foram mantidos sem alterações para preservar o contexto da época.
Podem conter redundâncias, limitações técnicas e implementações simplificadas.

Este material tem caráter histórico, com foco na evolução do conhecimento ao longo do tempo.
